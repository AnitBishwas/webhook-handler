# Swiss beauty webhook handler

**_This repo handles shopify webhooks in order to implment business logic and create segregated message in AWS SQS_**

**List of topics pushed through this repo currently available to subscribe to AWS SQS**

- ORDER_CREATE
- CASHBACK_PENDING_ASSIGNED
- CASHBACK_UTILISED
- ORDER_CANCEL
- CASHBACK_CANCEL
- ORDER_DELIVERED
- CASHBACK_ASSIGN
